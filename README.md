Usage
=====

`vagrant init senta/centos65-php55-lamp` then `vagrant up` to run Vagrant VM.


Description
===========

minimum LAMP Box for Vagrant

- CentOS 6.5
- PHP 5.5
- MySQL 5.1
- Apache 2.2.15

Requirements
============

- Vagrant
